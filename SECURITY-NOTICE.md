# 原始碼政策說明

本倉庫為 **輕度智能障礙學員面試訓練系統** 的公開展示用途，完整原始碼依以下原因不予公開：

1. **國科會計畫成果**：本系統為國科會補助計畫的研究成果，原始碼依計畫規範管理
2. **合作機構資料保護**：系統內含與台灣師範大學及合作特教機構相關的業務邏輯
3. **服務持續營運中**：系統目前仍在線上服務真實使用者，公開核心邏輯可能影響服務安全性

## 本倉庫包含

- 系統架構說明（`ARCHITECTURE.md`）
- 代表性程式碼片段（`snippets/`）
- 功能截圖（`screenshots/`）
- Demo 影片連結

## 本倉庫不包含

- 任何 API Key、資料庫密碼或憑證
- 可重建完整服務的關鍵程式碼
- 使用者個人資料
